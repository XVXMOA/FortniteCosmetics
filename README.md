# FortniteCosmetics

A web project for exploring and displaying Fortnite cosmetics.

This repository contains a React/TypeScript web app built with Vite, Tailwind CSS, and shadcn‑ui that lets users browse Fortnite cosmetic items (skins, emotes, backblings, etc). It’s designed to be a frontend UI for Fortnite cosmetic collections, themed around organization and visual clarity.

This project does **not** interact with Fortnite game servers directly and does **not** provide cheats, hacks, exploits, or ways to unlock items in the game itself. It’s meant for cosmetic data browsing and display only.

## 🚀 Features

- Built with **Vite**, **React**, and **TypeScript**.
- Styled using **Tailwind CSS** and shadcn‑ui components.
- Responsive and modern cosmetics catalogue UI.
- Structured for easy development and customization.

## 📦 Tech Stack

The project uses:

- **TypeScript**
- **React**
- **Vite**
- **Tailwind CSS**
